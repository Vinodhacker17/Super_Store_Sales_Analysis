# Super Store Sales Analysis 📊

An end-to-end Power BI dashboard analyzing retail sales, profit trends, and regional performance to drive data-driven decision making.

## 🚀 Dashboard Preview
![Overview](Screenshot%202026-03-25%20094014.png)
![Regional Analysis](Screenshot%202026-03-25%20094041.png)

## 🔍 Key Insights & Features
* **Sales Performance:** Tracked total revenue and profit across multiple years to identify seasonal peaks and growth trends.
* **Geospatial Analysis:** Used Map visualizations to pinpoint high-performing states and underperforming regions.
* **Product Breakdown:** Identified **Technology** and **Office Supplies** as the primary revenue drivers.
* **Profitability Tracking:** Monitored profit margins by sub-category to highlight areas with negative margins or high discounts.

## 🛠️ Tools Used
* **Power BI Desktop:** Data modeling and Visualization.
* **Power Query:** Data cleaning and transformation.
* **DAX:** Created calculated measures for KPIs.

## 📁 How to Use
1. Download the `Superstore_Sales_Dashboard.pbix` file.
2. Open it in Power BI Desktop to interact with the filters and charts.
